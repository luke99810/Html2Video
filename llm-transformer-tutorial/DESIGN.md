# LLM Transformer 教程视频 - 视觉设计规范

## 风格定位
**技术感 / 深色主题 / 数据可视化 / 教育科普**

适合讲解 AI/ML 底层原理，营造专业、前沿、清晰的视觉氛围。

## 色彩系统

### 主色
- **背景色**: `#0a0a0f` (深邃太空黑)
- **主强调色**: `#00d4ff` (科技青 - 用于标题、重点)
- **次强调色**: `#7c3aed` (紫罗兰 - 用于数据、图表)

### 辅助色
- **成功/正向**: `#10b981` (翠绿)
- **警告/注意**: `#f59e0b` (琥珀)
- **文字主色**: `#f8fafc` (纯白灰)
- **文字次色**: `#94a3b8` (中灰)
- **边框/分割**: `#1e293b` (深灰蓝)

### 渐变
- **主渐变**: `linear-gradient(135deg, #00d4ff 0%, #7c3aed 100%)`
- **辉光效果**: `0 0 40px rgba(0, 212, 255, 0.3)`

## 字体

### 标题字体
- **主标题**: `Inter`, `system-ui`, sans-serif, weight 800
- **章节标题**: `Inter`, weight 700

### 正文字体
- **解释文字**: `Inter`, weight 400
- **代码/数据**: `JetBrains Mono`, `Fira Code`, monospace

## 动画原则

### 入场动画
- 使用 `gsap.from()` 从屏幕外/透明状态进入
- 偏移起始时间 0.2-0.4s，避免同时出现
- 缓动函数: `power3.out` (主要), `expo.out` (强调), `back.out(1.2)` (活泼)

### 过渡效果
- 场景间使用淡入淡出或滑动过渡
- 持续时间: 0.6-0.8s
- 禁止在过渡前使用 exit 动画

### 数据可视化
- 使用 `scale` 和 `opacity` 展示数据流动
- 连接线使用 SVG stroke-dashoffset 动画
- 节点使用脉冲辉光效果

## 布局规范

### 安全区域
- 左右边距: 160px
- 上下边距: 120px
- 内容区域使用 flex column，gap 24-40px

### 网格系统
- 使用 CSS Grid 进行多列布局
- 常见比例: 1:1, 2:1, 3:2

## 禁止事项

- 不使用纯白色背景 (#ffffff)
- 不使用超过 3 种强调色
- 不使用低于 16px 的字号
- 不使用复杂的 3D 变换
- 不使用闪烁或快速频闪效果
