CHAPTER 01

为什么AI离不开数学？

AI = 优化问题
  • 训练神经网络就是在 最小化损失函数
  • 梯度下降法是优化的核心工具
  • 链式法则让反向传播成为可能
数据 = 数学对象
  • 文本、图像都是 向量和矩阵
  • 注意力机制是矩阵运算
  • Embedding 把语义压缩成向量
不确定性 = 概率
  • 模型输出是 概率分布
  • 贝叶斯推理处理先验知识
  • 交叉熵衡量预测与真实差距
信息 = 量化计算
  • 信息熵衡量不确定性
  • KL散度比较概率分布
  • 互信息评估特征相关性

CHAPTER 02 · CALCULUS

微积分：优化的数学

导数、梯度、链式法则 — 神经网络学习的基石

链式法则 (反向传播)
∂L/∂x = ∂L/∂u · ∂u/∂x
计算损失对每个参数的梯度
梯度下降
θ = θ - α · ∇θ J(θ)
沿梯度反方向更新参数
学习率调度
α_t = α₀ / (1 + β·t)
随训练逐渐衰减学习率
动量优化
v_t = β·v_{t-1} + ∇θ J(θ)
加速收敛，逃离局部最优

CHAPTER 03 · LINEAR ALGEBRA

线性代数：数据的语言

向量、矩阵、张量 — AI的底层表示

注意力机制
Attention(Q,K,V) = softmax(QKᵀ/√d)V
Transformer 核心计算
矩阵乘法
Y = W·X + b
神经网络层的前向传播
Embedding 查找
v_token = W[token_id]
词向量查表操作
Layer Normalization
y = (x - μ) / σ · γ + β
稳定训练的归一化操作

CHAPTER 04 · PROBABILITY

概率论：不确定性的数学

贝叶斯推断、概率分布、交叉熵

贝叶斯定理
P(A|B) = P(B|A)·P(A) / P(B)
融合先验知识与观测数据
交叉熵损失
H(p,q) = -Σ p(x)·log(q(x))
分类任务的标准损失函数
正态分布
N(x;μ,σ²) = exp(-(x-μ)²/2σ²) / √(2πσ²)
权重初始化和高斯噪声
最大似然估计
θ* = argmax Π P(x_i|θ)
从数据中学习概率参数

CHAPTER 05 · INFORMATION THEORY

信息论：信息的度量

熵、互信息、KL散度

信息熵
H(X) = -Σ p(x)·log₂ p(x)
衡量随机变量的不确定性
KL散度
D_KL(P || Q) = Σ P(x)·log(P(x)/Q(x))
衡量两个分布的差异
互信息
I(X;Y) = H(X) - H(X|Y)
衡量两个变量的相关性
困惑度
PPL = exp(-1/T Σ log P(w_t))
语言模型的核心评估指标
