CHAPTER 01

一般算法：编程的内功

排序、查找、递归 — 这些是所有 AI 的基石

quick_sort.py
1def quick_sort(arr):
2 if len(arr) <= 1:
3 return arr
4
5 pivot = arr[len(arr) // 2]
6 left = [x for x in arr if x < pivot]
7 middle = [x for x in arr if x == pivot]
8 right = [x for x in arr if x > pivot]
9
10 return quick_sort(left) + middle + quick_sort(right)
11
12# 测试
13print(quick_sort([3,1,4,1,5,9]))
💡 核心思想
  • 分治法：大问题分解为小问题
  • 递归：函数调用自身
  • 时间复杂度：平均 O(n log n)
  • 空间复杂度：O(n)

二分查找：高效的搜索

有序数组查找，时间复杂度 O(log n)

binary_search.py
1def binary_search(arr, target):
2 left, right = 0, len(arr) - 1
3
4 while left <= right:
5 mid = (left + right) // 2
6
7 if arr[mid] == target:
8 return mid # 找到了！
9 elif arr[mid] < target:
10 left = mid + 1 # 搜索右半部分
11 else:
12 right = mid - 1 # 搜索左半部分
13
14 return -1 # 没找到
📈 算法可视化
Step 1
定位中点
计算 mid = (L+R)/2
Step 2
比较大小
与 target 比较，决定区间
Step 3
缩小范围
L=mid+1 或 R=mid-1
查找 [1,3,5,7,9,11] 中的 7
→ 只需 3 步！

🧠 神经网络核心概念
  • Linear 层：y = Wx + b，矩阵运算
  • ReLU：max(0, x)，添加非线性
  • 损失函数：衡量预测与真实差距
  • 反向传播：计算梯度，更新参数
  • 优化器：Adam、SGD 等
