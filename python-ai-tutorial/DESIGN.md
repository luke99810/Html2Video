# Python AI 全栈教程 - 设计规范

## Style Prompt
教育类技术教程，代码编辑器风格，温暖但不花哨。清晰展示代码逻辑，步骤分明，适合小白跟学。重点突出"学完就能动手"的实操感。

## Colors
- **Background**: #1E1E2E (深紫灰，代码编辑器风格)
- **Surface**: #2D2D44 (卡片/面板背景)
- **Primary**: #89B4FA (蓝色，强调色)
- **Secondary**: #A6E3A1 (绿色，成功/代码关键字)
- **Accent**: #F9E2AF (黄色，高亮/警示)
- **Text**: #CDD6F4 (主文本)
- **Muted**: #6C7086 (次要文本)
- **Code String**: #F38BA8 (粉红，字符串)
- **Code Number**: #FAB387 (橙色，数字)

## Typography
- **Headings**: Inter, system-ui
- **Body**: Inter, system-ui
- **Code**: JetBrains Mono, Fira Code, monospace

## Motion Style
- 温和的滑入动画，不抢眼
- 逐行代码展示，有节奏感
- 场景切换使用淡入淡出

## What NOT to Do
- 不要用太花哨的渐变动画
- 不要堆砌多个动画同时执行
- 不要使用过于鲜艳的颜色组合
- 代码部分保持高可读性，不要过度装饰
